### 合并两个排序的链表

输入两个递增排序的链表，合并这两个链表并使新链表中的节点仍然是递增排序的。

示例1：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

#  先理解抽象他们的关系, 落地解决逻辑
```java

 public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if(l1 == null){
            return l2;
        }
        if(l2 == null){
           return l1;
        }
       ListNode root = null;
        ListNode e = null;
        while (l1 != null && l2 != null ){
            ListNode temp ;
            # 判断l1 和 l2 哪个值比较大, 保存值比较小的一份
            if(l1.val > l2.val){
                temp =  l2;
                l2 = l2.next;

            }else{
                temp =  l1;
                l1 = l1.next;
            }
            # 初始化情况下如果root
            if(root == null){
            	# 初始化root节点
                root = temp;
                # 保存入口节点
                e = temp;
            }else {
            	# 将较小的节点保存在节点尾部
                root.next = temp;

                root = root.next;
            }
            #判断如果l1 是空的话将l2 连接到尾部返回
            if(l1 == null){
                root.next = l2;
                continue;
            }
            #判断如果l2 是空的话将l1 连接到尾部返回
            if(l2 == null){
                root.next = l1;
                continue;
            }

        }
        return e;
    }

```
